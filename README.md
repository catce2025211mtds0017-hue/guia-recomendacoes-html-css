# Guia de Recomendações sobre Filmes de Fantasia

Este projeto é uma página criada para recomendar filmes do gênero cinematográfico **fantasia**, apresentando histórias que misturam magia, aventura e mundos extraordinários. A página indica ao leitor/espectador três obras importantes do gênero: **Wicked**, **a saga Harry Potter** e **Alice no País das Maravilhas (2010)**, destacando os elementos marcantes de cada filme e incentivando o público a conhecê-los.A página também apresenta uma tabela organizada com cada recomendação, links para sites confiáveis onde o leitor pode encontrar mais informações (como IMDb e AdoroCinema) e também indica em quais serviços de streaming cada filme está disponível.

## Tecnologias usadas
- **HTML5**
- **CSS**
